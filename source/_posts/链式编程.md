# 链式编程

本文介绍一下OC链式编程的基础使用，废话不多说，直接看代码

##示例类

```bash

@interface Person : NSObject


@property (nonatomic, copy)NSString *name;

@property (nonatomic, assign)NSInteger age;



- (Person *(^)(NSString *))xy_name;



- (Person *(^)(NSInteger))xy_age;

@end

```

```bash


@implementation Person


- (Person *(^)(NSInteger))xy_age{
    
    return ^(NSInteger xy_age){
        
        self.age = xy_age;
        
        return self;
        
    };
    
}


- (Person *(^)(NSString *))xy_name{
    
    return ^(NSString * xy_name){
        
        self.name = xy_name;
        
        return self;
        
    };
    
}


@end
	
```

##简单使用

```bash

Person *person = [[Person alloc]init];
    
person.xy_age(10).xy_name(@"张三");
    
NSLog(@"name = %@, age = %ld", person.name, person.age);

```
